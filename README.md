## Inspiration
The Ethereum (as well as greater blockchain community) is overrun with nonsubstantive trolling, and there is no place where this is more evident than Twitter. We setout to solve this problem via a Universal Basic Income-style economic incentive we call the â€œThought Leader Dividendâ€, administered via a DAO.

### Quick workflow

1. The `DAO` selects a `Humblee` to send `The Humble Potato` to. 

2. The `Humblee` has `2h` to pay the `Lottery Pool` which is set to `2% fee` of `Total Followers` (can be modified / ajusted by the `DAO`). For example, if the `Humblee` has 70,000 Followeers, the `donation` is set to `$1400` (70,000 * 0.02). 

3. If `Humblee` paies the `Donation` to the `Lottery Pool`, the smartcontract then triggers `Humiliation Mechanism` where the `Humblee` has to appology on Twitter. The `Humblee` can then signal another `Humblee` to send the potato to. But only the `DAO` has autorization to make final call. 

4. If the `Humblee` doesn't want to donate 2% fees to the pool, the smartcontract triggers the `Suffering Period`, `7 days` where people can join the `Lottery Pool` to unfollow the `Humblee`. 

5. At the end of the `Lottery Period`, the winner wins the `yield`, `interests` generated by the `Lottery Pool`. The Humble Potato expect to have a better yield than `http://compound.finance` has it should receive donations from the `Suffering Period` and the `Humiliation Period` mechanism.
